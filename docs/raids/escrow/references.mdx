## Previous (Split) Escrow

- Previous Raid Guild Escrow home page: https://smartescrow.raidguild.org/
- The previous version of the SmartEscrow contract was an iteration that used a single custom split interface in the escrow itself. This was a great learning experience, but we have since moved to using the [SplitMain](https://splits.org) contract for splitting funds.
- Direct link to a specific raid escrow: `https://smartescrow.raidguild.org/[raidID]`
